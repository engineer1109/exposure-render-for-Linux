# exposure-render-for-Linux  
Super Volume Render of Monte Carlo Path tracing for Linux  
## Introduction  
The code is mainly for volume render of Monte Carlo Path tracing running on Linux.  

## Dependency  
QT 5.14   (maybe 5.12+)  
CUDA 11.0  (maybe 10.0+)  
VTK 8.2  
GCC 9  

## BUILD && RUN  
```
mkdir build  
cd build
cmake ..  
make  
./ExposureRender  
```
